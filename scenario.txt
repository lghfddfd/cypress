1 - проверка слайдера
	1 шаг - пользователь заходит на главную страницу
	2 шаг - пользователь нажимает на кнопку прокрутки слайдера несколько раз
2 - поиск элемента
	1 шаг - пользователь заходит на главную страницу
	2 шаг - пользователь ищет текст "Карта лояльности"
3 - проверка разрешения
	1 шаг - пользователь заходит на главную страницу
	2 шаг - пользователь ищет текст "Карта лояльности"
4 - пользователь переходит в категорию готовая еда
	1 шаг - пользователь заходит на главную страницу
	2 шаг - пользователь нажимает на кнопку готовая еда
	3 шаг - пользователь нажимает на кнопку добавить в корзину
5 - заказ товара
	1 шаг - пользователь заходит на главную страницу
	2 шаг - пользователь нажимает на кнопку готовая еда
	3 шаг - пользователь нажимает на кнопку добавить в корзину	
6 - заказ товара
	1 шаг - пользователь заходит на главную страницу
	2 шаг - пользователь нажимает на кнопку готовая еда
	3 шаг - пользователь нажимает на кнопку добавить в корзину
	4 шаг - пользователь переходит в корзину
	5 шаг - пользователь проверяет наличие 'суп' в корзине
